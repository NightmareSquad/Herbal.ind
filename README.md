## Tips Meningkatkan CTR 100%

1. **Judul yang Menarik** – Gunakan kata kunci “Premium”, “100% Alami”, atau “Herbal” di atas fold.  
2. **Visual Produk Berkualitas** – Foto close-up, high-res, dan lifestyle shot (orang memakai / mengonsumsi).  
3. **CTA Jelas dan Warna Kontras** – Tombol “Beli Sekarang” harus terlihat berbeda dari background.  
4. **Testimoni dan Social Proof** – Sertakan review asli untuk membangun trust.  
5. **Segmentasi Audiens Meta Ads** – Target orang yang tertarik health, herbal, atau suplemen alami.  
6. **Pixel & Retargeting** – Pastikan Meta Pixel sudah terpasang untuk retarget pengunjung yang belum membeli.  
7. **Copywriting Ringkas tapi Mengena** – Sorot manfaat utama dalam 3–4 poin saja di atas fold.

## Cara Deploy ke GitHub

1. Buat repository baru dengan nama **herbal.ind**  
2. Upload semua file landing page (HTML, CSS, JS, gambar, dsb.) ke repository  
3. Aktifkan **GitHub Pages**:  
   - Settings → Pages → Branch `main` → `/root` → Save  
   - Dapatkan URL landing page: `https://username.github.io/herbal.ind/`  
4. Gunakan URL ini sebagai link tujuan di Meta Ads
